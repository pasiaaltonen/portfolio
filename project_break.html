Break-Fast Derby

Break-Fast derby is a miniature car racing game where player competes AI cars in various different settings. Currently the prototype has five different race tracks for singleplayer experience and two tracks for splitscreen multiplayer.


Game is in 3D with slightly angled top down view. Tack themes consists of dinner table, sandbox, miniature forrest setting, snooker table and a warehouse setting. Each track is slightly different from another. For multiplayer there are dinner table track and sandbox track available. Current version contains only quick races with one available car. So, other drivable cars are not available.
